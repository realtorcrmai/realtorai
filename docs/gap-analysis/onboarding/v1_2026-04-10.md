# Gap Analysis: Signup & Onboarding — v1

**Date:** 2026-04-10 | **Analyst:** Claude + Rahul | **Completion:** ~35%
**PRD Reference:** `docs/PRD_Signup_Verification_Onboarding.md`, `docs/SPEC_Realtor_Onboarding_v2.md`
**Benchmarks:** Monday.com, Follow Up Boss, Real Geeks, Lofty, HubSpot, Wise Agent, KVCore, Sierra Interactive, Salesforce

---

## Methodology

7-pass gap analysis comparing Realtors360's current implementation against:
1. **Monday.com** — best-in-class SaaS onboarding (visual wizard, card selection, celebration moments)
2. **Follow Up Boss** — best real estate CRM onboarding (email-first integration, admin-first sequencing)
3. **Industry best practices** — SaaS onboarding research (HubSpot, Userpilot, ProductLed data)

Each gap is rated: **Have** (fully implemented), **Partial** (exists but incomplete), **Missing** (not implemented).

### Biggest Gaps (Highest Impact)

| Priority | Gap | Impact | Effort | Benchmark |
|----------|-----|--------|--------|-----------|
| **P0** | Signup too many fields (S1, S6) | ~30% conversion loss | Small | Monday.com: 2-3 fields |
| **P0** | No celebration/fireworks (C1, C2, C3) | Emotional engagement, retention | Small | Monday.com: confetti + staggered checkmarks |
| **P0** | No personalization wizard (P1-P9) | All users get same generic experience | Medium | Monday.com: 4-question visual wizard |
| **P1** | No persistent checklist (PO1) | Users forget incomplete setup tasks | Medium | Monday.com: bottom-right widget. FUB: 10-step page |
| **P1** | No free trial (B1, B3) | Users can't experience premium features | Small | FUB: 14-day trial |
| **P1** | No AI in onboarding (A1-A7) | Missed #1 differentiator | Medium | No competitor has this — we can own it |
| **P2** | No empty state design (PO3) | Blank pages kill activation | Medium | Monday.com: illustration + CTA per page |
| **P2** | No welcome drip emails (D1-D6) | No re-engagement for drop-offs | Medium | FUB: Success Manager + email sequence |
| **P2** | No email-first integration (I1) | Miss FUB's killer move: auto-populate from inbox | Large | FUB: connect email → CRM fills itself |
| **P3** | No CSV import in onboarding (O5) | Established agents can't import | Small | FUB: CSV with field mapping |
| **P3** | No in-app video tutorials (PO4) | Slower learning curve | Medium | FUB: 1-3 min contextual videos |
