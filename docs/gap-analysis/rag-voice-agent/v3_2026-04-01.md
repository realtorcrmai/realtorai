## Recommendations for Next Phase

1. **Prompt injection classifier** — Fine-tune Haiku to classify injection attempts. Regex catches 80% but misses creative rephrasing. Classifier adds defense-in-depth.
2. **Query expansion** — Implement HyDE (Hypothetical Document Embedding): generate a hypothetical answer, embed it, search for similar docs. Improves recall on vague queries.
3. **Endpoint audit** — Systematically review all 21 voice-agent routes + 10 RAG routes for tenant isolation consistency.
